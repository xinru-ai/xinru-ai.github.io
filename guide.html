首次设置教程

这份教程恢复自你之前写过的说明，并按网站页面结构重新整理。目标是让新用户第一次打开心如时，能按照最稳妥的顺序把它真正用起来。

开始前先知道

当前测试版主要用于朋友试用、早期测试和 demo 展示。
如需使用云端 AI，建议保持联网。
如果要使用 Whisper Small、Whisper Medium 或本地 AI，请预留更多磁盘空间。
当前优先支持 Apple Silicon Mac；未来其他平台会逐步补充。

系统要求

macOS 12 或更高版本
Apple Silicon Mac：M1 / M2 / M3 / M4
建议先准备一个可用的云端 AI key
如果要使用本地 AI，请预留额外磁盘空间

添加云端 AI 来源

这是最建议新用户先完成的一步。先把云端来源配置好，心如会更快进入真正可用的状态。

打开首页右上角的 设置，进入 AI配置
点击右上角的 添加来源
在新建来源面板中选择 云端
选择服务商，例如 OpenAI 或 DeepSeek
填写来源名称，把你的 API key 粘贴到 API Key 栏
检查 Base URL 和模型是否正确
点击 测试连接，通过后再点击 保存

为什么建议先做这一步：它是最快让心如真正开始工作的方式；如果不先设置云端来源，很多人设一开始就不会有可用的默认来源。

下载本地 AI 模型

打开 `设置 -> 本地AI管理`
浏览当前可用的本地模型
根据自己的需求选择一个模型
点击 `下载`
等待模型下载完成

本地模型下载完成后，它会自动出现在 `AI配置 -> 默认本地来源` 中。对这些由系统管理的本地模型，你不需要再手动新建一个本地来源。

选择默认来源

回到 `设置 -> AI配置`
先选一个 `默认云端来源`
再选一个 `默认本地来源`

为什么这一步重要：心如的人设依赖默认来源；新用户一开始不会自动拥有默认来源；如果跳过这一步，有些人设就可能无法正常工作。

语音识别模型建议

`Whisper Medium`：如果你更重视识别质量，优先推荐
`Whisper Small`：对很多用户来说是比较平衡的选择
`Whisper Tiny`：更适合更在意速度、对准确率要求没那么高的用户
更小模型更快更轻；更大模型更慢但准确率更高

快速使用建议

不确定选哪个人设时，先从 `通用优化` 开始
中英翻译时，使用 `翻译模式`
整理会议内容时，使用 `会议记录`
想让 AI 直接回答问题时，使用 `通用AI` 或 `助手模式`
