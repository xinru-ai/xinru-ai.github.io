新手教程

第一步：添加云端 AI 来源

对于大部分电脑而言，运行本地大模型并不轻松。因此更建议新用户先完成云端大模型配置，让心如尽快进入真正可用的状态。

什么是 API Key？

API Key 可以理解为你在云端 AI 服务商那里的"使用凭证"。心如本身不卖这个 Key，也不会替你收这笔钱；费用是你直接支付给服务商，例如 DeepSeek、OpenAI 等。

如果你是第一次接触，可以先从 DeepSeek 开始。注册、充值后就能拿到自己的 Key，再把它填进心如里即可。通常入门成本并不高，少量充值就能使用很长一段时间。

前往 DeepSeek 获取 API Key

第二步：下载本地 AI 模型

如果你的电脑性能和磁盘空间允许，可以继续下载一个本地 AI 模型。这样后面除了云端来源，你也会拥有可离线使用的本地来源。

第三步：选择默认来源

这一步很简单：选一个默认本地来源，再选一个默认云端来源。之后大多数人设就能按预期工作。

第四步：下载语音识别模型

语音识别模型负责把你的声音转成文字。M 系列用户推荐 Whisper Medium，追求速度可选 Whisper Small 或 Tiny。

语音识别模型建议

`Whisper Medium`：如果你更重视识别质量，优先推荐
`Whisper Small`：对很多用户来说是比较平衡的选择
`Whisper Tiny`：更适合更在意速度、对准确率要求没那么高的用户
更小模型更快更轻；更大模型更慢但准确率更高

快速使用建议

不确定选哪个人设时，先从 `通用优化` 开始
中英翻译时，使用 `翻译模式`
整理会议内容时，使用 `会议记录`
想让 AI 直接回答问题时，使用 `通用AI` 或 `助手模式`
